Setup Instructions
==================

1.  Visit http://www.python.org/download/releases/2.7.4/ and download 
    the appropriate file for your operating system (Windows/Mac/Linux).
    Use all the default/standard options for installation.

2.  If your terminal size is already 80x24 or 80x25, skip this step.
    (The default terminal size on Windows is 80x25, on Mac/Linux it
    is 80x24)

    If you would like to check:
    Open the game folder and run "calib.py". Follow the instructions.

3.  On Windows, double-click the "Les Misérables Game" link.
    On Mac/Linux, run "LesMisGame".
