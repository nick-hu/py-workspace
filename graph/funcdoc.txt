==========================
Func Classes Documentation
==========================

class func.Poly(coefs)
    Polynomial function

    coefs is a list of coefficients of the form [a, b, c, ..., d, e],
    where f(x) = ax^n + bx^(n-1) + cx^(n-2) + ... + dx^1 + e

    f(x) = 2x^3 - 5x + 5 --> Poly([2, 0, -5, 5])

class func.PolyTrans(deg, trans)
    General transformation polynomial function

    deg is the degree of the function

    trans is a list of transformations of the form [k, a, b, h],
    where f(x) = a(b(x-h)) ^ deg + k

    If trans has length zero, f(x) = x^deg

class func.Abs(f)
    Absolute value function

    f is a function for which a(x) = |f(x)|

class func.Combined(f, g, op)
    Combined function (includes rational functions)

    op is an operation '+', '-', '*', '/', or '@'

    f and g are functions for which
    c(x) = f(x)+g(x) or f(x)-g(x) or f(x)*g(x) or f(x)/g(x) or f(g(x))

class func.Radical(f)
    Radical function

    f is a function for which r(x) = sqrt(f(x))

class func.Exponent(trans)
    General transformation exponential function

    trans is a list of transformations of the form [k, c, a, d, h],
    where f(x) = ca^(d(x-h)) + k

    If trans has length one, it is a list [a], where f(x) = a^x

class func.Log(trans)
    General transformation logarithmic function

    trans is a list of transformations of the form [k, c, a, d, h],
    where f(x) = cloga(d(x-h)) + k

    If trans has length one, it is a list [a], where f(x) = loga(x)

class func.Trig(f, trans)
    General transformation trigonometric function

    f is a trigonometric function
    'sin', 'cos', 'tan', 'asin', acos', 'atan',
    'sinh', 'cosh', 'tanh', 'asinh', 'acosh', or 'atanh'

    trans is a list of transformations of the form [a, b, c, d],
    where f(x) = af(b(x-c)) + d

    If trans has length zero, f(x) = f(x)

class func.Point(d)
    Point "function"

    d is a dictionary of points of the form {x1: y1, x2: y2, ...}
    where (x1, y1), (x2, y2), ... are ordered pairs of a function

    Note: Functions must have only one element in the range for
    every element in the domain (i.e., {1: 2, 2: 4} is acceptable,
    but {1: 2, 1: 3} is not).
